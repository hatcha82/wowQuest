선행 퀘스트 : 인형 재료 (http://wow.somegate.com/topic.php?topic_uid=31140)

퀘스트 임무 : 가라앉은 사원=아탈학카르 신전에 있는 트롤들에게서 '부두 깃털' 6개를 모아 '임프시'에게 가져다준다.

퀘스트 레벨 : 52 Lv (?)

퀘스트 보상 : 경험치 + 돈(1골드 55실) + 아이템 3가지중 1개 선택


아탈학카르 신전에 있는 트롤들을 잡아서 3종류(황색,청색,녹색)의 부두깃털을 각각 2개씩 가져가면 되는데,
마이잔, 줄로, 로로, 게이셔, 후쿠, 졸로를 처치하라고 임프시가 힌트를 줍니다.


아탈학카르 신전에 있는 모든 트롤들이 주는 것이 아니라 저주받은 전당에 있는 6명의 네임드 트롤들만 준다는 것이죠.


로로(2시) : 녹색 부두 깃털
줄로(3시) : 황색 부두 깃털
게이셔(4시) : 황색 부두 깃털
후쿠(7시) : 청색 부두 깃털
졸로(9시) : 녹색 부두 깃털
마이잔(11시) :  청색 부두 깃털

황색 : 게이셔, 줄로
청색 : 후쿠, 마이잔
녹색 : 로로, 졸로





영혼 수확기 : 지팡이(양손), 체력+16, 착용효과(주문의 적중률이 1%만큼 증가, 암흑 계열의 주문과 효과의 공격력 최대 24만큼 증가)


나락의 조각 : 장신구, 사용효과(마나나 영혼의 조각을 사용하지 않고 보이드워커를 소환합니다)


예속의 로브 : 가슴방어구(천), 체력+15, 지능+10, 착용효과(화염계열과 암흑계열의 주문과 효과의 공격력 최대 23만큼 증가)

마지막으로...퀘스트 완료하시고 바로 자리를 뜨지 마시고..임프시와 나이비의 썰렁 개그쇼를 잠깐 구경하고 가시길...
