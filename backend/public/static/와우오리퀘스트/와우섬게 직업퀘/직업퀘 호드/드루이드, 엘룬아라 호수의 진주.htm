안녕하십니까?

월드 오브 워크레프트를 사랑하는 게임 메니아 여러분

제가 퀘스트를 수행하기위해 여기저기 사이트를 찾아보았지만 직업퀘스트는 나와있는 곳이 없더군요

그리하야 제가 하나 올려볼까합니다

1차퀘스트 : 썬더 블로프의 장로의 봉우리의 상급드루이드에게 퀘스트를 부여받음

                 달의숲 엘룬아라호수 중앙의 깊은 부분에서 진주가 든 궤짝 발견  >> 달의 숲의 텐드리트 스타블레이즈 에게 퀘스트완료

2차퀘스트 : 바다표범의 시험 (민첩선의, 인내의 바다펜던트조각)

   >>>>>  민첩성의 바다펜던트 >> 불모의땅 북동쪽 진흙늪의 중앙에위치한 이상한 상자를 열어 펜던트 획득

   >>>>>   인내의 바다펜던트 >> 언더시티 공동묘지 가기전에서 서쪽으로가면 해변에 부서진 배한척 >> 여기서 서쪽으로 수영해가다보면(주의!! 호흡)

                  수심이 긒어지는 지점에 수중 바닥에 갈라진 바위틈에서 공기방울이나오는 곳이 있을것이다. 여기서 는 수중에서 호흡이 가능하다.

                 하지만 펜던트를 휙득하기 위해서는 더깊은곳으로 들어가야하기때문에 마음 단단히 먹고 준비하시길

                 이곳에서 또 서쪽으로 깊은 바다로 들어가다보면 상자 하나가 보일것이다. 여기서또 주의할점은 항상 호흡에 신경을 쓰되 만일 상자를 찾지 못하였는데

                 호흡이 반을 넘어선다면 다시 바위틈으로 돌아가 숨을 가다듬고 다시 서쪽 방향으로 상자가 보이고 상자를 열고 펜던트를 획득 

                 그러나 호흡이 다되어 바위틈 공기방울 있는 곳까지 갈수없는 사람도있을것이다 이럴때는 먼저 상자위치를 먼저 확인한뒤 호흡을 바위틈에서하고

                  다시 상자를 열때까지 일직선경로가되어야하며 조금이라도 상자를 찾기위해 돌아다녔다면 죽을 각오는 하는것이 좋음

   >>>>>  달의숲 북서쪽에 위치한 제단에서 펜던트를 합친후   달의 숲의 텐드리트 스타블레이즈 에게 퀘스트완료  

3차퀘스트 : 썬더 블레이드의 장로의봉우리에있는 상급드루이드 에게가서 최종 퀘스트를 완료하면

                  바다표범으로 변신 할수 있는 스킬을 배울수있다 >> 효능 : 수영을할때만 변신이 가능하며  바다표범으로 변신을 했들때는 물속에서 호흡을 할수 있다

                  단!! 바다표범으로 변신 하였을 경우는 다른 스킬을 사용 못하며 물 밖으로 나오게되면 자동으로 변신은 풀리게된다~~~~

제가 이 퀘스트를 수행하는데 얼마나 힘들었는지 모르실꺼에요~~ ㅜㅡ 우엉

바다표범으로 변신하고나니 물속에서 수영이 더 빨라지고 물속에서도 호흡걱정할 필요 없어서 참 좋았습니다

여러분들도 부디 퀘스트 수행해서 수중에서 호흡걱정 하지 말자구요~~ ^^*

지금까지 줄진 서버의 피어싱드루 였습니다~~~ ^^*
